# Installation
### Download and change folder
```
cd ~
git clone https://github.com/ahoeppner/dotfiles.git

cd dotfiles
```
### Run install (Linux only - does not work on OSX)
```
./install.sh
```

# Uninstallation

### Change to dotfiles folder
```
cd ~/dotfiles
```

### Run uninstaller
```
./uninstall.sh
```
